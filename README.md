## 1. Instruções para executar a aplicação
O projeto contem um arquivo Makefile com os comandos para executar a aplicação. Para utiliza-lo, é necessário executar os comandos à seguir:

1. `rmiregistry`
2. `make compile`
3. `make run-server`
4. `make run-client`
